# ExoryNotes

ExoryNotes is a simple, lightweight, and powerful note-taking app for Android.

## Features
- Create and manage notes and checklists
- Add images and audio recordings to notes
- Export notes as PDF, TXT, JSON, or HTML
- Set reminders for notes
- Label and organize notes
- Archive and trash management
- Auto backup support
- Customizable text size and theme
- Widget support
